SQL Clauses & Operators

 Objective
The purpose of this task is to practice SQL clauses and operators used for filtering, comparing, and manipulating data in a database. These queries help retrieve specific records and perform calculations on stored data.

 Topics Covered

- DISTINCT Clause
- Column Aliases (AS)
- WHERE Clause
- Comparison Operators (`>`, `<`, `>=`, `!=`)
- Arithmetic Operators (`+`, `-`, `*`, `/`)
- Logical Operators (`AND`, `OR`, `NOT`)
- NULL Conditions (`IS NULL`, `IS NOT NULL`)
- IN and NOT IN
- BETWEEN and NOT BETWEEN
- LIKE and NOT LIKE
- Combined Conditions

Learning Outcomes

After completing these SQL queries, you will be able to:

- Retrieve unique records using `DISTINCT`.
- Rename column names using `AS`.
- Filter records using the `WHERE` clause.
- Compare values using comparison operators.
- Perform arithmetic calculations within queries.
- Apply logical conditions using `AND`, `OR`, and `NOT`.
- Handle missing values with `IS NULL` and `IS NOT NULL`.
- Filter records using `IN` and `NOT IN`.
- Select ranges of values using `BETWEEN`.
- Search text patterns using `LIKE` and `NOT LIKE`.
- Combine multiple conditions to create complex queries.
